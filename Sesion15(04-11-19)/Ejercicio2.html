First-line

Se suelen utilizar para dar estilos a unas partes específicas de un elemento. Pseudoelemento first-line: se utiliza para dar estilo a la primera línea de un elemento. Sólo se puede aplicar a elementos de bloque.

First-letter

Se utilizar para dar estilo a la primera letra de un elemento. Sólo se puede aplicar a elementos de bloque. La primera letra tiene otro tamaño y otro estilo de fuente.
